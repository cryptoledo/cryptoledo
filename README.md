## What I'm Building

**COGNIS·NEXUS** — cognitive middleware that gives AI agents self-awareness, long-term memory, judgment, and a conscience before they act.

Every AI agent today is powerful but unaccountable — it acts immediately, forgets everything, and can't explain itself. I built the layer that sits between an agent and the world: scoring confidence, running an internal critic, storing episodic memory, and checking policy before any tool call executes. Every action is logged, signed, and auditable.

Built on the **MALSE™ architecture** (patent pending) — five cognitive phases mapped from human brain function to AI agent software:

| Phase | Human Analogy | What It Does |
|-------|--------------|--------------|
| Metacognition | Self-awareness | Scores confidence + hallucination risk per output |
| Attention (Critic) | Devil's advocate | Challenges every uncertain answer before delivery |
| Learning (Memory) | Long-term memory | Remembers every session, every mistake, every win |
| Social Cognition (ToM) | Social awareness | Agents model what other agents know and need |
| Executive Function (CGA) | Conscience | 7-step check before every consequential action |
